Esse projeto visa estruturar uma aplicação que simule um blog com o conteudo focado em pescaria.

//tecnologias 
A aplicação usa da tecnoloia Next.js para estruturar as funcionalidades do App Router e SEO: com rotas dinamicas, Server Components, SSR, criação de metadados dinamicos e com deploy no Vercel.

//funções 
A aplicação permite visualizar em lista ordenada posts ficticios de um blog de pescaria, com paginas para filtrar por posts em alta e posts recentes, cada post pode ser visualizado separadamente por meio de rotas dinamicas do App Router.
